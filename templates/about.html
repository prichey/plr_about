About this site

Hosted on Heroku.

Written in Python with the Flask microframework.

Uses pyLast to interface with the last.fm API and OpenPaths to sync with the app on my phone for GPS location (still hoping nobody uses this maliciously).

099 uses the flickrapi module to scrape photos from the corresponding photo set on Flickr. gbmf uses the same, and adds a hyperlink to the latitude and longitude metadata from the picture via Google Maps.

Currently the script freshly scrapes images each time it is run. I'm working on implementing a database / cron job to expedite the process.
